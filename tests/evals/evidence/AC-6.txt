DECISION
--------
The real LLM run (tier L2/L3 with live ANTHROPIC_API_KEY calls) is deliberately
descoped from this release cycle by explicit decision. It is NOT executed in this
cycle and is NOT a blocking gap. The real-spend evals work will be addressed later
in a dedicated evals framework, scoped as a separate project. Treat AC-6's state
as DEFERRED (deliberate), not PENDING.

Rationale
---------
- Running L2/L3 burns real Anthropic token budget; the deliberate decision is to
  consolidate all live-LLM evaluation into a purpose-built evals framework rather
  than running an ad-hoc subset inside this release cycle.
- The CI on-ramp (subset selection + budget-guarded CI job) is already wired and
  remains in place for that future work; only the real-spend execution is deferred.

------------------------------------------------------------------

- pyproject.toml — added `ci_subset` to [tool.pytest.ini_options] markers.
- tests/layer2_llm_evaluation/test_agent_behavior.py — two single-filter routing
  tests marked @pytest.mark.ci_subset (the cheapest end-to-end L2 signals):
    TestRoutingBehavior::test_terraform_routing

    * runs `pytest tests/layer2_llm_evaluation -m ci_subset -q`
