Oracle property (why this is an oracle, not a regression detector)
------------------------------------------------------------------
Every expected_decision is curated by a human against security-tiers policy
and the enforcement anchors in hooks/modules/security/{tiers,mutative_verbs,
blocked_commands,approval_grants}.py. It is NOT extracted from a recorded log
(contrast tools/gaia_simulator extractor.py:330 expected_decision=outcome
['decision'], which makes that path a regression detector). Decision
vocabulary = Claude Code PreToolUse permissionDecision {allow, ask, deny}.
Coverage: T0 read-only, T1/T2 validate/plan, local-safe writes (git add/commit,
gaia ac add), T3 mutations (git push, kubectl/terraform apply, gaia approvals
approve), consent-reducing exception (gaia approvals revoke -> allow),
flag-dependent (curl -X POST, git branch -D), dangerous composition
(curl | sh), and never-approvable deny (rm -rf /, dd of=/dev/sda).
